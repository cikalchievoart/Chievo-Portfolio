ESP32-Based Multi-Sensor Temperature and Humidity Monitoring Simulation Integrated with Blynk

The “Multisensor Room Prototype” project is an Internet of Things (IoT)-based room monitoring and automation system powered by the Espressif Systems ESP32 microcontroller and simulated using the Wokwi platform. The system integrates DHT22, HC-SR04, and LDR sensors to monitor temperature, humidity, object distance, and room light intensity. Developed using C++ programming, the system automatically controls LED indicators and displays monitoring data on a 16x2 LCD based on real-time sensor conditions. The data is also transmitted via Wi-Fi and synchronized with the Blynk IoT cloud platform for remote monitoring through an interactive dashboard.
